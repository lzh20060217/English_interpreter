# AI 智能同传助手 - 外网访问设置

## ✅ ngrok 已配置完成

### 📋 下一步操作

**双击运行这个文件：**
```
启动ngrok.bat
```

### 🎯 会发生什么

1. **弹出一个黑色窗口**（ngrok 服务）
2. 等待几秒钟（可能需要 5-10 秒）
3. 在该窗口中会显示类似内容：
   ```
   Session Status                online
   Account                      your@email.com
   Forwarding                   https://abc123xyz.ngrok-free.app -> http://localhost:3000
   ```

### 🌐 如何访问

1. 复制 `Forwarding` 后面的 **https://** 地址
   - 例如：`https://abc123xyz.ngrok-free.app`

2. 在**任意设备**（手机、平板、电脑）的浏览器中打开这个地址

3. 就可以使用 AI 智能同传助手了！

### ⚠️ 注意事项

- **保持 ngrok 窗口打开**：关闭后服务会中断
- **免费版限制**：
  - 每次重启会生成不同的地址
  - 4小时内免费使用
  - 带宽有限制
- **麦克风权限**：首次使用时，浏览器会询问是否允许使用麦克风，选择"允许"即可

### 🔧 如果遇到问题

1. **无法连接**：检查是否在同一时间运行了多个 ngrok
2. **权限错误**：确保以管理员身份运行
3. **Token 过期**：可能需要重新配置 token

### 📱 测试建议

推荐测试步骤：
1. 在手机上打开 ngrok 地址
2. 点击"开始录音"
3. 给予麦克风权限
4. 对着手机说话
5. 查看是否能正常录音
